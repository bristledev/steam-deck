#  External Resources (Keep Learning!)

You've made it through the entire series! You now have a solid foundation for using, managing, and optimizing your Steam Deck like a pro. But the world of SteamOS and Linux is vast, and there's always more to discover.

Here's a curated collection of the best places to continue your journey — organized by the topics we've covered.

---

## 🎮 Steam Deck & SteamOS

### Valve Official
- **[Steam Deck Official Site](https://www.steamdeck.com/)**: Valve's landing page with specs, features, and the latest announcements.
- **[SteamOS Recovery Instructions](https://help.steampowered.com/en/faqs/view/1B71-EDF2-EB6D-2BB3)**: Valve's official guide for creating a recovery USB and re-imaging your Deck.
- **[Steam Deck FAQ](https://www.steamdeck.com/en/faq)**: Valve's official answers to the most common questions.
- **[Deck Verified](https://www.steamdeck.com/en/verified)**: How Valve's compatibility rating system works.

### Community
- **[r/SteamDeck (Reddit)](https://www.reddit.com/r/SteamDeck/)**: The largest collection of Steam Deck owners. Great for news, show-off posts, and troubleshooting.
- **[Steam Deck Discord](https://discord.gg/steamdeck)**: A very active real-time community for specific technical issues.
- **[Steam Deck HQ](https://steamdeckhq.com/)**: Dedicated to testing games and providing "best settings" for performance.

---

## 🐧 Linux & The Terminal

### Shells
- **[GNU Bash Manual](https://www.gnu.org/software/bash/manual/)**: The complete, official reference for Bash — the default shell on SteamOS.
- **[Fish Shell Documentation](https://fishshell.com/docs/current/)**: The official docs for Fish, including its tutorial for beginners.
- **[GNU Readline Documentation](https://tiswww.case.edu/php/chet/readline/rltop.html)**: The full reference for all those keyboard shortcuts we covered.

### Core Tools
- **[GNU Coreutils Manual](https://www.gnu.org/software/coreutils/manual/coreutils.html)**: The complete guide to `ls`, `cp`, `mv`, `chmod`, and every other fundamental command.
- **[Explainshell.com](https://explainshell.com/)**: Paste any command and get a plain-English breakdown of every argument.
- **[cheat.sh](https://cheat.sh/)**: Quick, community-driven cheat sheets for any command, right from your terminal (`curl cht.sh/tar`).
- **[tldr pages](https://tldr.sh/)**: Simplified, practical command examples — a friendlier alternative to `man` pages.

### Learning Linux
- **[Linux Journey](https://linuxjourney.com/)**: A beautifully designed, free curriculum that takes you from zero to comfortable with Linux.
- **[The Linux Command Line (Free Book)](https://www.linuxcommand.org/tlcl.php)**: William Shotts' legendary free book — the gold standard for learning the terminal.
- **[Arch Wiki](https://wiki.archlinux.org/)**: SteamOS is built on Arch Linux, so this wiki is directly relevant. It's the most comprehensive Linux wiki in existence.

---

## 📦 Package Management & App Installation

- **[Flatpak Documentation](https://docs.flatpak.org/)**: The official docs for understanding how Flatpak apps work, permissions, and sandboxing.
- **[Flathub](https://flathub.org/)**: The main app store for Flatpak apps — browse everything available from a web browser.
- **[Homebrew Documentation](https://docs.brew.sh/)**: The official guide for the Homebrew package manager.
- **[Nix Manual](https://nix.dev/)**: The best starting point for learning Nix (more approachable than the official NixOS manual).
- **[NixOS Wiki](https://wiki.nixos.org/)**: The community-maintained wiki for all things Nix.
- **[AppImageHub](https://www.appimagehub.com/)**: A directory of apps distributed as AppImages.

---

## 🔧 Gaming & Proton

- **[ProtonDB](https://www.protondb.com/)**: Community compatibility reports for every Steam game on Linux. Filter by Steam Deck for the most relevant results.
- **[Proton GitHub](https://github.com/ValveSoftware/Proton)**: Valve's open-source Proton repository — issue tracker, release notes, and changelogs.
- **[GE-Proton (Glorious Eggroll)](https://github.com/GloriousEggroll/proton-ge-custom)**: The community Proton fork with extra patches and codec fixes.
- **[ProtonUp-Qt](https://davidotek.github.io/protonup-qt/)**: The graphical tool for managing Proton and Wine versions.
- **[Protontricks](https://github.com/Matoking/protontricks)**: Install Windows components into individual game prefixes.
- **[Heroic Games Launcher](https://heroicgameslauncher.com/)**: Play your Epic Games and GOG libraries on Linux.
- **[EmuDeck Wiki](https://emudeck.github.io/)**: The definitive source for emulation setup on the Steam Deck.
- **[Decky Loader](https://decky.xyz/)**: The plugin framework for Game Mode customization.
- **[SteamGridDB](https://www.steamgriddb.com/)**: Custom artwork for your game library.

---

## 🌐 Networking & Remote Access

- **[OpenSSH Documentation](https://www.openssh.com/manual.html)**: The official manual for SSH — the protocol powering wireless file transfers.
- **[WinSCP Documentation](https://winscp.net/eng/docs/start)**: Guides for the Windows SFTP client we recommended.
- **[Tailscale Documentation](https://tailscale.com/kb/)**: Excellent, beginner-friendly guides for setting up remote access.
- **[Tailscale on SteamOS](https://tailscale.com/kb/1011/install-steamos/)**: Tailscale's specific guide for the Steam Deck.

---

## 🐳 Containers & Sandboxing

- **[Podman Documentation](https://docs.podman.io/)**: The official Podman docs — rootless containers without a daemon.
- **[Podman Desktop](https://podman-desktop.io/)**: A graphical interface for managing containers (if you prefer clicking over typing).
- **[Distrobox GitHub](https://github.com/89luca89/distrobox)**: The official repo with guides for running full Linux distros inside containers.
- **[Distrobox Docs](https://distrobox.it/)**: The project's dedicated documentation site.

---

## 🐍 Scripting & Development

- **[Python.org Tutorial](https://docs.python.org/3/tutorial/)**: The official Python tutorial — the best place to start learning.
- **[Real Python](https://realpython.com/)**: High-quality, practical Python guides for all skill levels.
- **[GitHub Docs](https://docs.github.com/)**: Official documentation for Git and GitHub.
- **[Git - The Simple Guide](https://rogerdudler.github.io/git-guide/)**: A no-nonsense, one-page introduction to Git.
- **[systemd Documentation](https://www.freedesktop.org/software/systemd/man/latest/)**: The official reference for services, timers, and units.
- **[Arch Wiki: systemd](https://wiki.archlinux.org/title/Systemd)**: A more approachable guide to systemd, directly relevant to SteamOS.

---

## ✨ Power User Tools

- **[Starship Prompt](https://starship.rs/)**: Official site with installation, configuration, and preset guides.
- **[zoxide GitHub](https://github.com/ajeetdsouza/zoxide)**: The smarter `cd` command — documentation and usage examples.
- **[fzf GitHub](https://github.com/junegunn/fzf)**: The fuzzy finder — full documentation and integration guides.
- **[Fish Shell Plugin Manager (Fisher)](https://github.com/jorgebucaran/fisher)**: Manage Fish plugins easily.

## Final Advice: Be Fearless!

The single best resource is your own curiosity. Because of tools like **[Distrobox](https://github.com/89luca89/distrobox)** and **[Podman](https://podman.io/)**, you can experiment without fear.

Don't be afraid to break things (in a container!), try new shells, or customize the UI until it's perfect. That's the beauty of the Steam Deck — it's not just a console; it's *your* Linux PC.

**Happy Gaming!** 🎮
